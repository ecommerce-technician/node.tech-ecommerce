# node.tech
## Creates a simple node app with angular

### Simple install
```
sh launch.sh
```

### Manual install
```
xcode-select –-install
ruby -e “$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)”
brew doctor
brew install caskroom/cask/brew-cask
brew intstall git
brew install node
git pull
npm install
npm install -g bower
bower install
gulp scripts
gulp sass
forever app.js
```
